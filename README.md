# blackjack-python
A Beginner Python blackjack capstone project.
This is a terminal-based blackjack game built in Python.

## Features
- Player vs Dealer gameplay
- Random card dealing
- Score calculation
- Win/lose conditions

## How to Run
Run the Python file in your terminal:

python blackjack.py

## Concepts Used
- Functions
- Loops
- Conditionals
- Random module
